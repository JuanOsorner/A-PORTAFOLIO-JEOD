Bienvenido a **Proyectos JEOD**, un repositorio centralizado que muestra mi trayectoria de desarrollo, habilidades técnicas y proyectos de software. Esta colección sirve como portafolio de mi trabajo en desarrollo full-stack, automatización y aprendizaje continuo.

## Estructura del Repositorio

Este repositorio está organizado en tres secciones principales, cada una con un propósito distinto:

### 1. [Proyectos](Proyectos/README.md)
Contiene aplicaciones principales y proyectos de desarrollo integrales.
-   **Java**: APIs de nivel empresarial usando Spring Boot (`familia-api`).
-   **Python**: Integraciones de IA (`chat_gemini`), aplicaciones web (`Django_pdfs`) y práctica de frameworks (`practica_ej2`).

### 2. [Inventario](Inventario/Scripts/README.md)
Una colección de scripts de utilidad, herramientas de automatización y ejercicios de programación.
-   **Scripts**: Automatización en Python para procesamiento de datos (Excel, JSON).
-   **Pruebas**: Pruebas de concepto (POCs) para decoradores, logging y gestión de entornos.

### 3. [Certificaciones](Certificaciones/README.md)
Documentación de logros profesionales y certificaciones técnicas obtenidas en:
-   Git y GitHub
-   Programación en Python
-   Ecosistema React.js

## Primeros Pasos

Para explorar cualquier proyecto localmente:

1.  **Clonar el repositorio**:
    ```bash
    git clone https://github.com/JuanOsorner/proyectos_JEOD.git
    ```
2.  **Navegar** a la carpeta del proyecto específico (por ejemplo, `cd Proyectos/Python/chat_gemini`).
3.  Seguir las instrucciones específicas del **README.md** dentro de cada carpeta para su instalación y ejecución.
