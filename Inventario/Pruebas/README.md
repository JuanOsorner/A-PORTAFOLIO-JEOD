# Pruebas y Prácticas de Python

Este directorio contiene una colección de scripts y pequeños proyectos diseñados para practicar y reforzar conceptos de Python. Aquí se exploran temas como decoradores, manejo de logs, variables de entorno y manipulación de archivos.

## Lista de Proyectos

A continuación se enumeran los proyectos y ejercicios contenidos en este directorio:

1.  **p_decoradores**:
    -   Ejercicios prácticos centrados en el uso de decoradores en Python.
    -   Abarca desde niveles básicos hasta avanzados.
    -   Incluye ejemplos de validación de argumentos y lógica matemática (álgebra lineal).

2.  **probando_logging**:
    -   Pruebas de configuración del módulo `logging` de Python.
    -   Ejemplos de cómo cargar y utilizar variables de entorno desde archivos `.env` usando `python-dotenv`.

3.  **prueba_dotenv**:
    -   Scripts complementarios para probar la lectura y escritura de archivos.
    -   Uso de `logging` para registrar eventos y errores durante la ejecución.
